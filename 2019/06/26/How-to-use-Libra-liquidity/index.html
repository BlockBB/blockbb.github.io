如何加入Libra呢？

在Facebook宣布创立Libra项目之后，我猜测关注Crypto领域的人有三种心理：

  1. 加速研究，步Facebook的后尘，开发自己的区块链产品和生态，快步加入Crypto领域。
  2. 观望着、犹豫着、等待监管政策，这通常都是大厂，人出名了，包袱就多，瞻前顾后，他们或许忘了自己当年从小团队成长起来的时候，反而是第一个吃行业🦀️的人，所以从这点来说，FB值得敬佩，敢于挑战旧世界。
  3. 最后一种是现实主义者，也是当前Crypto领域的创业者，如何加入Libra生态？如何利用Libra生态的 Liquidity Value？是他们所思考的，这篇日记主要讨论这个话题。

有意思的是，在Libra白皮书出来后不久，我看到朋友圈的一位创业者朋友就抛出来这个话题，计划筹集资金，加入Libra。

Wow！这就是创业者的态度，看来第三种人还真不孤独。

Libra 需要哪些团队进入呢 ？

WhitePaper里已经给定了硬性条件：市值十亿美金和一千万美金的会费，市值10亿美金的人拿出一千万跟玩一样，假设国际上有100个满足资金条件的申请者，那么Libra会看重哪些条件呢？从现在列出来的企业可以看出个大概。

一是有钱、有人、有政策支持的银行，反正是可以帮其流转、兑换Libra的，有了他们相当于有了政策的支持，同时他们也是最难拉拢的人，所以这时候第二种公司就尤为重要。

二是有海量用户的机构，例如Uber类型的，Visa类型的支付机构。 但Uber里的大部分用户可能对cytpto并不感冒，他们也只有使用的需求，在金融系统发达的欧美地区，打个车、点个外卖用信用卡就完了，怎么吸引他们是个问题。

从第二个条件来推断，如果我有千万精准的、对Libra有需求的用户，并且这些用户有持续的需求，而且这些用户分布在Facebook App 触及不到的人群，例如中国微信用户、日本的Line用户、韩国的Kakao用户、东南亚地区的用户、阿拉伯地区、拉美、南非等国，我想Libra基金会会乐于跟这样的项目合作，如果前期FB的重心在这些国外地区，那么面临美国的监管压力可能也会小一些，这些用户对Facebook本身也是个有益的用户扩充。

有了用户方向，再来看看这些用户聚集在哪里呢？ 回过头来看我这位朋友的需求，首先产品达到10亿美金的市值，并且有海量用户和持续的使用需求，这两个条件必须全部满足才能吸引Libra的注意，同时满足Libra和这位创业者需求的业务也就逐渐清晰起来了 — 加密货币交易所

任何一个团队加入Libra，都想利用其海量用户资源，Coinbase不可能想不到这一点，他现在的用户集中在美国，未来呢？交易所的用户天然对Libra的稳定货币有需求，只要有可能，他们就会持有，即使Libra要求严格的KYC，也会有大量用户进行注册，外国政府根本禁止不了，退一步来说，假如交易所的大部分用户不能满足Libra 的Kyc需求也没关系，只要交易所本身通过KYC，持有大量的Libra就可以交易了，甚至未来会有针对Libra的稳定币在交易所里进行兑换，只要这个交易所进行信用背书就可以，都说USDT不安全，还不是这么多人在使用，Libra的信用度肯定比USDT高，个人看法，无论Facebook基于何种目的发币，Crypto领域的创业者都应尽量利用它的影响力和流动性价值，抱有怀疑的态度是没问题，但不应该体现在创业行动上。

关于Libra的稳定币，让我大胆猜测下，估计下面的事情都会发生（不是预测，预测要给出时间，给出出现的可能性）：

  1. 针对Libra的稳定币会出现
  2. 针对Libra的OTC会出现
  3. FB会逐渐开放对加密货币交易所的支持，例如开放个API给交易所，不过话说回来，即使FB不开放它的用户资源，加入LIbra的交易所也应该像办法吸引FB的用户群体，办法肯定是有的。。
  4. 会有针对Libra的借贷服务，Libra本身有点跨国央行的意思，有M0，慢慢就会产生M1、M2（虽然我也不知道金融系统里的M012345678是咋算出来的，哈！），Libra的稳定币影响力会逐层递加，必定产生一些金融衍生服务。

5.Libra会在虚拟世界里被率先应用，例如买个音乐、游戏、虚拟服务付费 还有购买加密货币等领域率先发力，而购买加密货币会是最快发生的，加密货币领域的用户会是对Libra最友好的的一批人，现在交易所里的稳定币服务未来都会被引入到Libra里，不管Facebook愿不愿意，政府不想看得到BTC繁荣发展，不还是一样禁止不了。

交易所是加密货币里的排头兵，是接触加密货币的人群首先要使用的服务，这也是我在那篇总结币安的日记里着重阐述的。

回头再说为什么交易所最能满足这位创业者朋友的要求，首先交易所最容易聚集用户，也有明显的盈利模式，并且最容易爆发，你看看17年的Binance，19年的爆发的二线交易所，都印证了这种情况，其次交易所不但可以通过手续费来获取利润，还可以利用众筹从有资金实力的人手里募集需要的子弹，放眼加密货币领域，这是最好的方式了，我相信这位创业者朋友，会认真思考此方式的。

未来某一天，当Libra公布会员的招收标准的时候，交易所领域内有远见的团队都会跃跃欲试为此准备，在Libra正式推出之前，留给这些有远见的人的时间越来越少了，未来只会竞争更激励。

结语

在我跟伙伴们做加密货币技术布道的时候，一直在思考如何更好的拓宽加密货币人群，如何把这些理念和技术推广出去，如何引导众人进入这个迷人的领域，为他们提供何种服务等，所以当我看到朋友圈有人讨论Libra的时候，就引出了这篇文章所讨论的话题，不过又跟交易所扯上了关系，也是挺有意思的，到现在为止，我陆续写了3篇跟交易所有关的日记，其中的两篇如下：
